Demo drag-drop, geser kotak merah dengan mouse/jari

Ada banyak cara untuk menggunkan Blitz Edu. Salah satunya adalah dengan mendownload template dan mengeditnya.

Anda bisa mendownload template disini: [template.js]

Kode minimal dari BE adalah sebagai berikut: 

Grafis(240, 320);

let spr = Muat("./gbr/box.png", true);
Posisi(spr, 160, 120);

function Loop() {
    Bersih();
    Gambar(spr);
}

Penjelasan tiap baris adalah sebagai berikut:

Grafis(240, 320);

Ini adalah pintu masuk dari BE. Fungsi ini harus dipanggil di awal, untuk mengeset kanvas, dan mengatur semuanya agar bisa berjalan dengan baik dan benar.
Fungsi ini memiliki dua parameter utama: panjang dan lebar.
keduanya menentukan panjang dan lebar kanvas.
Parameter sisanya bisa Anda baca di sini

let spr = Muat("./gbr/box.png", true);

Fungsi Muat() berfungsi untuk memuat gambar dan mengkonvertnya ke dalam bentuk Sprite.
Sprite adalah tokoh utama dari BE. Tanpa Sprite maka tidak akan ada yang ditampilkan di layar.
Parameter pertama adalah URL, bisa lokal ataupun online.
Bila Anda meload gambar dari online, pastikan Anda mendapatkan dari yang punya gambar. Parameter kedua dari fungsi ini adalah untuk menentukan apakah Sprite bisa di drag atau tidak.
Drag-Drop adalah fitur bawaan dari Sprite.
BE fokus pada interaksi, dan sepertinya drag-drop adalah feature yang bakal banyak dipakai. Maka dari itulah Saya memutuskan bahwa drag-drop harus ada secara default.

Posisi(spr, 160, 120);

Fungsi ini untuk menentukan posisi Sprite. Parameternya adalah object Sprite dan posisi X serta Y. 

function Loop() {
    Bersih();
    Gambar(spr);
}

Fungsi Loop() adalah fungsi untuk perulangan, akan dipanggil 60 kali per detiknya (60 fps) -- tergantung kecepatan device. Dalam fungsi ini kita memanggil dua fungsi lainnya:

Bersih()

Sesuai namanya, maka fungsi ini akan membersihkan layar.

Gambar(spr)

Sesuai namanya juga, fungsi ini akan menggambar di layar. Parameternya adalah Sprite yang akan digambar. Sprite akan digambar pada posisi yang telah ditetapkan sebelumnya.
