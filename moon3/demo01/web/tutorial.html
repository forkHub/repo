Demo drag-drop, geser kotak merah dengan mouse/jari

Ada banyak cara untuk menggunkan Blitz Edu. Anda bisa mencoba secara daring, mendownload javascript, meng-klone dari git, atau dengan mendownload template dan mengeditnya.

Anda bisa mendownload template disini: tempalate js. Template ini untuk javascript. Bagi pengguna typescript bisa mendownload template untuk typescript, melalui tautan yang tersedia di halaman depan.

Bila anda buka file index.html maka isinya adalah sbb:

BE hanyalah kumpulan fungsi (pustaka/library), bukan framework ataupun enggine, sehingga tidak membutuhkan struktur yang rumit. Yang terpenting adalah keberadaan element canvas. Jika seandainya elemen kanvas tidak tersedia pun juga tidak mengapa, karena BE akan membuatnya secara otomatis.

Selanjutnya jika anda buka file Drag.js, maka isinya sebagai berikut:

Grafis(300, 300);

let spr = Muat("./gbr/box.png", true);
Posisi(spr, 150, 100);

function Loop() {
	Bersih();
	Gambar(spr);
	Tulis("Kotak ini bisa di drag", 300 / 2, 300 / 2);
}

Penjelasan tiap baris adalah sebagai berikut:

Grafis(300, 300);

Ini adalah pintu masuk dari BE. Fungsi ini harus dipanggil di awal, untuk mengeset kanvas, dan mengatur semuanya agar bisa berjalan dengan baik dan benar.
Fungsi ini memiliki dua parameter utama: panjang dan lebar.
keduanya menentukan panjang dan lebar kanvas.
Selain kedua parameter ini, masih ada parameter yang lain, yang bisa Anda baca secara lebih detail di bagian daftar perintah.

let spr = Muat("./gbr/box.png", true);

Fungsi Muat() akan memuat gambar dan mengkonvertnya ke dalam bentuk Sprite.
Sprite adalah tokoh utama dari BE. Tanpa Sprite maka tidak akan ada yang ditampilkan di layar.
Parameter pertama adalah URL, bisa lokal ataupun online.
Bila Anda meload gambar dari online, pastikan Anda mendapatkan ijin dari yang punya gambar. Parameter kedua dari fungsi ini adalah untuk menentukan apakah Sprite bisa di drag atau tidak.
Drag-Drop adalah fitur bawaan dari Sprite.

Posisi(spr, 150, 100);

Fungsi ini untuk menentukan posisi Sprite spr pada posisi 150 dan 100. 

function Loop() {
    Bersih();
    Gambar(spr);
}

Fungsi Loop() adalah fungsi untuk perulangan, dan akan dipanggil 60 kali per detiknya (60 fps) -- tergantung kecepatan device.
Dalam fungsi ini kita memanggil dua fungsi lainnya:

Bersih()

Sesuai namanya, maka fungsi ini akan membersihkan layar.

Gambar(spr)

Sesuai namanya juga, fungsi ini akan menggambar di layar. Parameternya adalah Sprite yang akan digambar.

Tulis("Kotak ini bisa di drag", 300 / 2, 300 / 2);

Menulis pada posisi 150, 150

Demikian penjelasan singkat dari contoh aplikasi sederhana ini. Anda bisa membaca secara detail tentang masing-masing fungsi pada bagian daftar perintah di halaman depan, atau dengan melihat pada contoh-contoh yang sudah disediakan.
