Basik Blok

Basik Blok (BBlok) is a visual code playground that uses drag-and-drop blocks powered by Blockly. The syntax is inspired by Blitz Basic

You can use BBlok to create a simple canvas based application. The result can be published and embeded in other website as an iframe.

You can use BBlok as an online editor or download it for local use. Download the offline version here
