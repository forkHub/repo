# Basic Blockly

Basic Blockly (BBlockly) is a visual code playground that uses drag-and-drop blocks powered by [Blockly](https://developers.google.com/blockly/). 
The syntax is inspired by [Blitz Basic](https://en.wikipedia.org/wiki/Blitz_BASIC).

You can use BBlockly to create simple canvas based application. 

You can use BBlockly to create a simple canvas based application.
The result can be published and embeded in other website as an iframe.

You can use BBlockly as an online editor or download it for local use.
