Blitz Edu [Draft]

Blitz Edu (BE) adalah kumpulan fungsi javascript untuk menangani kanvas.
BE dibuat untuk tujuan pendidikan, membuat berbagai peraga digital, dan pembelajaran kode bagi pemula.
Sintak BE dibuat sederhana dan tidak membutuhkan arsitektur yang rumit.
BE bisa dipakai untuk programmer pemula dan mahir.
